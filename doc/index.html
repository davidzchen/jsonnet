A more elegant config language

Jsonnet is a domain specific configuration language that helps you define JSON data. Jsonnet lets you compute fragments of JSON within the structure, bringing the same benefit to structured data that templating languages bring to plain text.

Provenance, Copyright, License

Jsonnet was designed and implemented at Google as a 20% project. The name Jsonnet is a portmanteau of JSON and sonnet, pronounced "jay sonnet". Jsonnet has been open-sourced under the Apache 2.0 license. Jsonnet is not an official Google product (experimental or otherwise), it is just code that happens to be owned by Google.
